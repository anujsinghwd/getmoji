[Deno](https://deno.land) module starter repository.

## Usage

```typescript
    import { getEmojiByName, emojify } from "https://deno.land/x/getmoji@1.2/mod.ts";

    await getEmojiByName("pizza"); //return üçï

    await emojify("i love $pizza") //returns i love üçï
```
